# Nexus-AI

Nexus AI is an all-in-one career preparation platform featuring an AI-driven Resume Checker and an intelligent Interview Chatbot. It helps job seekers optimize their resumes for ATS systems and master their interview skills through personalized, role-specific mock sessions.

## 🚀 Features
- **AI Resume Analyzer**: Upload your resume to get instant, actionable feedback to improve its ATS compatibility.
- **Intelligent Chatbot UI**: A sleek chat interface to practice interview questions or ask career-related queries.
- **Modern Dashboard**: An interactive sidebar-based navigation system with a sleek React-based UI.

## 🛠 Tech Stack
- React
- TypeScript
- Vite
- SCSS Modules

## ⚙️ Getting Started

### Prerequisites
- Node.js installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/royrishika12/Nexus-AI.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Nexus-AI
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
